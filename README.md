# Healthcare-Shift-Handoff-AI-Assist

Built with React Native and Expo mobile building suite.

To run, clone the repo
```bash
cd frontend; npm i;
```
only ever run this in /frontend
```bash
npx expo start
```
It will build, then
```bash
› Press a │ open Android
› Press i │ open iOS simulator
› Press w │ open web
```
